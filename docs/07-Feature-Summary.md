### 五、键盘快捷键 ⌨️

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Space` / `空格` | 标记为认识 | 快速标记已掌握 |
| `Enter` / `回车` | 标记为不认识 | 添加到复习列表 |
| `N` / `n` | 下一个单词 | 跳过当前单词 |
| `S` / `s` | 打开设置 | 快速打开设置弹窗 |
| `A` / `a` | 生成AI例句 | 快速生成例句 |
| `ESC` | 关闭弹窗 | 关闭设置弹窗 |

**智能判断**：
- 输入框中不触发快捷键
- 设置弹窗打开时只响应ESC

## 🎯 快速开始指南

### 安装依赖
```bash
cd D:\my-projects\vocab-context
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 访问应用
浏览器打开：http://localhost:3005/

### 配置AI功能
1. 点击右上角"⚙️ 设置"
2. 输入硅基流动API密钥
3. 添加兴趣标签（可选）
4. 保存设置

### 开始学习
1. 点击"认识"或按空格键标记已掌握
2. 点击"不认识"或按回车键添加到复习
3. 点击"✨ 生成AI例句"获取个性化例句
4. 点击"⭐"收藏难词到单词本
5. 使用快捷键提升效率
