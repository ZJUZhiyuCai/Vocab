# VocabContext - 语境词汇学习工具

> 基于真实语境 + AI个性化例句 + 间隔重复的雅思词汇学习系统

**在线体验**: https://scintillating-dragon-3ef634.netlify.app/

## 项目简介

VocabContext 是一个基于CEFR分级的智能词汇学习系统，专注于雅思词汇的高效学习。通过语境学习、AI个性化例句和科学的间隔重复算法，帮助学习者从"认识"到"精通"。

### 核心特性

- 📚 **精选词库**：13,670个精选词汇，按难度分为5个级别
- 🎯 **语境优先**：真实语料提取的例句，贴近实际应用
- 🤖 **AI个性化**：基于学习目的生成定制化例句（需配置API）
- 🎲 **智能随机**：真正的随机学习，避免顺序偏差
- ⏰ **间隔重复**：科学的复习算法，长期记忆巩固
- 🎨 **莫兰迪配色**：温柔护眼，长时间学习不疲劳
- 💾 **本地优先**：数据本地存储，隐私完全可控
- 📱 **完美响应式**：桌面端和移动端无缝体验

### 技术亮点

- **全量加载**：一次性加载所有词库，启动快速
- **Fisher-Yates随机**：真正的随机分布
- **智能缓存**：AI内容自动缓存24小时
- **模块化词库**：按考试级别科学分段
- **PWA支持**：可安装到桌面，支持离线使用

---

## 技术栈

- **前端框架**: Vue 3 (Composition API) + Vite
- **UI样式**: Tailwind CSS (莫兰迪色系)
- **AI服务**: 硅基流动 API (Qwen2.5-72B-Instruct)
- **动画效果**: canvas-confetti
- **数据存储**: LocalStorage
- **部署平台**: Netlify

## 配置说明

### AI 功能配置

1. 打开应用，点击右上角 **⚙️ 设置**
2. 在 **🤖 AI功能** 区域输入硅基流动 API 密钥
3. 在 **🎯 学习目的** 中选择你的学习目的
4. 保存设置

**获取免费API密钥**:
- 访问 https://docs.siliconflow.cn/cn/userguide/quickstart
- 注册账号（免费）
- 创建 API 密钥

### 学习模式配置

- **每日学习目标**: 5-100词/天（默认20词）
- **学习模式**:
  - 顺序学习：按词库顺序学习
  - 随机学习：打乱顺序，避免顺序记忆（推荐）

## 技术支持

### 常见问题

**Q: AI功能如何使用？**
A: 在设置中输入硅基流动API密钥，即可使用AI例句功能。

**Q: 如何切换词库？**
A: 点击侧边栏的词库名称，或使用词汇水平测试功能。

**Q: 复习列表为什么有些单词有⚠️标记？**
A: 这些是"不认识"的单词，会优先复习。

**Q: 数据会丢失吗？**
A: 所有数据保存在浏览器本地，清除浏览器数据会丢失。建议使用云端同步备份。

**Q: 如何云端同步？**
A: 在设置中配置 GitHub Token，即可启用 GitHub Gist 同步。
